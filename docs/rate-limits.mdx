## Subscription Tiers

不同方案的并发限制如下：

| Tier | Rate Limit (RPM) | Features |
| :--- | :--- | :--- |
| **Starter** | 60 RPM | 基础 API 访问 |
| **Pro** | 300 RPM | 优先支持, 审计日志 |
| **Scale** | Custom | 多密钥管理, 专属带宽 |

*RPM = Requests Per Minute (每分钟请求数)。*
