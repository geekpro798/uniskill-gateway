## One Key. Infinite Skills.

UniSkill 为你的 AI 代理提供即插即用的现实世界能力。通过一个统一的 API，你的 Agent 即可解锁联网搜索、深度网页抓取以及社交媒体分析功能。

### 核心特性

* **永久积分 (Lifetime Credits)**: 只要订阅有效，积分永不过期，支持余额自动结转。
* **Agent 原生**: 专门为自主代理（Autonomous Agents）优化的返回格式（JSON/Markdown）。
* **极简集成**: 无论你使用 OpenClaw 还是 LangChain，只需一个 API Key 即可启动。
