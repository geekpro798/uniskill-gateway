## Credit System Overview

UniSkill 使用积分制计费，确保你只需为实际使用的能力付费。

### Weight Table (权重表)

| Skill | Description | Cost |
| :--- | :--- | :--- |
| `uniskill_search` | 全球联网搜索 | 10 pts / call |
| `uniskill_scrape` | 深度网页抓取 | 20 pts / page |

### Rollover Policy (结转政策)

UniSkill 的核心优势在于 **"Credits never expire"**。
只要你的订阅（Starter, Pro 或 Scale）保持活跃，所有未消耗的积分将自动滚动到下一个计费周期，永久有效。
